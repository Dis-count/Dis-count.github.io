## addMVar vs addVar

For MVar, 

z = m.addMVar(shape = W, vtype=GRB.CONTINUOUS)

just use z[0] to indicate the first variable.

For Var,

x ={}

for i in range(1,3):
    x[i] = model.addVar(lb = 0, ub = GRB.INFINITY, vtype = GRB.CONTINUOUS, name = 'x_' + str(i))

Use x[1] to indicate the first variable.
